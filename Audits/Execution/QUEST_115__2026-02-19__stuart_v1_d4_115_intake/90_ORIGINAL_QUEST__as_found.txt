This quest makes transcript selection real by adding transcript_version_id to the canonical run param contract and ensuring the FORMALIZE plan step includes it.

Update the door-facing run request contract to include transcript_version_id:

  - tolerate legacy aliases if any exist (optional)

- Update plan_builder.build_plan(...):

  - Do NOT change other params (mode/template_id/retention/speakers)

Add minimal validation in router.validate if needed:
