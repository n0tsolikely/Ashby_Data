## API CONTRACT (PROPOSED)
`GET /api/sessions/{session_id}/export?format=zip&export_type=full_bundle&transcript_formats=txt,pdf&formalization_formats=pdf,md`

Rules:
- `export_type=dev_bundle` ignores format selection and always exports DEV_EXPORT.
- `export_type` in `{full_bundle, transcript_only, formalization_only}` exports USER_EXPORT.
- Defaults (if params missing):
  - transcript_formats = `txt`
  - formalization_formats = `pdf`

## IMPLEMENTATION PLAN
1) Update `api_session_export()` in `app.py`:
   - accept `export_type == dev_bundle`
   - parse/validate `transcript_formats` and `formalization_formats`
   - pass these into the new export builder function introduced in QUEST_172

2) Zip filename rules:
   - Use `Content-Disposition` filename like:
     - `<session_id>__<sanitized_title>__export_<export_type>__<YYYYMMDD_HHMMSS>.zip`
   - Determinism note:
     - Timestamp component must be derived from stable source for deterministic runs OR be omitted for deterministic mode.
     - If you include a timestamp, it will break hash determinism. (Dungeon 8 wants determinism.)

**Recommended deterministic filename**:
- `<session_id>__<sanitized_title>__export_<export_type>.zip`
- If title missing: omit.

3) Keep `format!=zip` response behavior:
