Quest Diff Audit Files (Receipt Standard): This folder contains immutable audit bundles for completed quests. Each completed quest must have a matching audit bundle under QUEST DIFF AUDIT FILES/Execution/QUEST_###__AUDIT/ that includes terminal-style receipts and diffs. Quest completion is not valid unless the quest file references its audit bundle path (and the bundle exists). Minimum required receipts: (1) 01_DIFF.patch showing code/state changes, and (2) 03_TEST_OUTPUT.txt or equivalent execution evidence. Optional receipts include before/after terminal logs, file lists, and notes. Audit bundles are append-only receipts: do not overwrite; create a new bundle if a quest is reworked.
