CONTEXT / WHY
Dungeon 7 requires the LLM to receive an annotated transcript, not a raw blob:
- speaker-turn segments
- stable segment_id anchors
- timestamps
- speaker_label + mapped speaker_name (if available)

It also requires template shaping:
- raw template text
- parsed “template sections/schema” representation

Right now:
- Engine sends only transcript_text + mode/template_id/retention to the gateway.
- Gateway schema doesn’t accept segments or template content.

This quest upgrades the engine ↔ gateway contract *without changing the prompt yet*.
(Next quest upgrades the provider prompt.)
