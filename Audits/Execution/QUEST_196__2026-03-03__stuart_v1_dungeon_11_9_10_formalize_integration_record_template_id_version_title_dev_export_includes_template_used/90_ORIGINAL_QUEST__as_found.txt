SYSTEM CONTEXT STATEMENT
- Dungeon 11 requires that formalizations and exports keep traceability:
  - Every formalization output records:
    - template_id
    - template_version
    - template_title (display metadata)
  - Dev exports include the **exact template_text + metadata** used for each formalization
  - User exports do NOT include template internals (only id/title in session metadata)
- Current state:
  - minutes_json / journal_json already write template_id and template_version to output metadata
  - template_title is not yet included
  - export bundling does not include template internals
