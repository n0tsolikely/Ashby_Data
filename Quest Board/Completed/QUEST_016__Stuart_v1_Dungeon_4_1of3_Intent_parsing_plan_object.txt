DESCRIPTION

Implement the meetings module router interface that converts natural language + attachments + UI state into a structured intent and an executable plan. This quest defines the plan object and minimum intents.

SCOPE / OBJECTIVE
────────────────────────────────────────
Implement a meetings module router that accepts:
- user text
- attachment metadata (if present)
- UI selections (mode/template/speakers) when applicable
- session context (active session, last run)

Outputs (minimum):
- structured intent
- executable plan (ordered steps)

Minimum intents (v1), matching Guild Orders:
- set mode
- set speakers
- intake (store upload into session)
- formalize
- search
- export
- speaker map overlay request
- extract-only outputs request

This quest defines the plan structure; clarification and gating are handled in QUEST_017/018.
