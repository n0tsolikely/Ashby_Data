SYSTEM CONTEXT STATEMENT
========================================
If chat is going to be the interface, it cannot bullshit.

Dungeon 10 truth rules (must be enforced in code, not vibes):
- For meeting-derived claims, cite the archive (segment anchors).
- If there is insufficient evidence, say so (and optionally return relevant hits).
- In session scope, never 'helpfully' pull evidence from other sessions.
- In global scope, disclose when evidence is from a different session than current focus.

This quest adds enforcement + graceful degradation so the LLM cannot silently hallucinate facts.
