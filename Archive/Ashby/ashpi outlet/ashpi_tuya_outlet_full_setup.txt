4. OUTLET CONTROL SCRIPT (ashpi_plug_test.py)

Save this entire script as ashpi_plug_test.py on the Pi:

from env import ENDPOINT, ACCESS_ID, ACCESS_KEY, USERNAME, PASSWORD, DEVICE_ID
from tuya_iot import TuyaOpenAPI, TUYA_LOGGER
import logging

# Optional: enable debug logging
logging.basicConfig(
    format="%(asctime)s - %(name)s - %(levelname)s - %(message)s",
    level=logging.INFO,
)
TUYA_LOGGER.setLevel(logging.INFO)

# Initialize Tuya OpenAPI client
openapi = TuyaOpenAPI(ENDPOINT, ACCESS_ID, ACCESS_KEY)

# Country code "1" = US/Canada, "smartlife" is the app schema.
openapi.connect(USERNAME, PASSWORD, "1", "smartlife")


def set_plug(on: bool):
    """Turn the Tuya plug on/off."""
    commands = {
        "commands": [
            {"code": "switch_1", "value": on}
        ]
    }
    res = openapi.post(f"/v1.0/devices/{DEVICE_ID}/commands", commands)
    print("Response:", res)
    return res


def main():
    print("Press Enter to turn plug ON...")
    input()
    set_plug(True)

    print("Press Enter to turn plug OFF...")
    input()
    set_plug(False)


if __name__ == "__main__":
    main()


5. HOW TO USE

Each time you want to quickly test/control the outlet from the Pi:

1) SSH into the Pi and activate the virtual environment:

    ssh notsolikely@<your_pi_ip>
    source ashpi-env/bin/activate

2) Run the plug test script:

    python3 ashpi_plug_test.py

3) Follow prompts:
   - Press Enter once -> plug turns ON
   - Press Enter again -> plug turns OFF


6. WHAT TO CHECK IF IT BREAKS

- If Tuya returns sign / auth errors:
  * Confirm ACCESS_ID and ACCESS_KEY are unchanged in both env.py and Tuya cloud.
  * Confirm USERNAME and PASSWORD are correct Smart Life login (same account linked in Tuya).
  * Confirm ENDPOINT is still https://openapi.tuyaus.com and your project is in Western America.
  * Confirm DEVICE_ID matches the plug in Tuya's cloud device list.

- If Python says ModuleNotFoundError:
  * You probably forgot to activate the venv or install tuya-iot-py-sdk.
  * Run:
        source ashpi-env/bin/activate
        pip install tuya-iot-py-sdk
