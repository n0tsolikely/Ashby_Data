SECTION 4 — INTENT SCHEMA (NEW FILE 1)
------------------------------------------------------------------------

ALL NLU engines must output dictionaries like:

{
  "type": "lights.adjust",
  "group": "thor",
  "brightness": 800,
  "delta": null,
  "mode": "max"
}

Supported intent types:
- lights.on
- lights.off
- lights.adjust
- comfort.cold
- chat (fallback)
- unknown (local engine can't interpret)


------------------------------------------------------------------------
SECTION 5 — LOCAL NLU ENGINE (NEW FILE 2)
------------------------------------------------------------------------

Runs offline, instant, ~0 cost.

Detects:
- Group names (captain, thor, sky)
- “turn on” / “turn off”
- “darker / brighter / more / less”
- Max-mode (“full blast”, “max”, “all the way”)
- Comfort phrases (“cold”, “freezing”, “chilly”)

Returns an intent or "unknown" → triggers GPT fallback.

Cost per request:
